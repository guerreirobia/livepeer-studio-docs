---
title: Sample Apps
---

The following are lists of sample apps using Livepeer Studio features and capabilities:

| Sample App                                                                                                | Description | 
| -------------------------------------------------------------------------------------------------------------- | ---------- | 
| [Mint NFT](https://github.com/codentell/video-nft-sample)  |  Learn how to mint a video NFT                   |                                                                             |
| [Create a Video Player](https://github.com/livepeer/livepeer-jwplayer-demo) |     This is a sample app to showcase how to use Livepeer Studio APIs to create a live stream and play with JWPlayer. The app enables the user to create a stream and play it back via JWPlayer.     |                                        |
| [WebRTMP](https://github.com/livepeer/webrtmp-sdk ) |  JavaScript SDK for streaming media via RTMP from the Web. Originally designed for Livepeer.com, but can be used for any other service by running your own webrtmp-server. | 
| [Justcast.it](https://github.com/victorges/justcast.it) | Straightforward web app for 1-click starting a live stream |
| [Livepeer Web API Proxy](https://github.com/victorges/livepeer-web-api-proxy) | Proxy for limited access to the Livepeer Studio API for the web. | 
| [Livepeer NFT Gate](https://github.com/livepeer/livepeer-nft-gate-example) | Livepeer NFT Gate sample app |
