# Create a Livestream

This guide provides instructions on how to create streams from the Dashboard or
the API.

<!--tabs-->

# Dashboard

1. Log in to the Livepeer Studio Dashboard
2. Navigate to the Streams page, livepeer.studio/dashboard/streams. On this
   page, you can create a new stream and delete streams. When you name your
   stream, pick a unique name and include URL-compatible characters only.
3. On the Stream page (livepeer.studio/dashboard/streams), you can also click on
   a stream to find out configuration and usage information.
